博客搭建

hexo init 可能会因为网络原因失败，多试几次

  • 上传Github

    在E:/hexo-jason-blog 右键打开Open Git Bash here终端命令行页面，执行命令hexo clean && hexo g && hexo d

  • 建立文章
    在E:/hexo-jason-blog 右键打开Open Git Bash here终端命令行页面，执行命令hexo [layout] '文章名字'

添加菜单配置

  • Home（首页）、Archives（目录）、Gallery（相册）、About（关于我）、Search（搜索）五部分，其中博客选择的主题中Gallery（相册）、About（关于我）、点击提示没有被找到，需要的话可进行配置

  • About（关于我）

    桌面右键打开Open Git Bash here终端命令行页面，进入D:\Blog路径执行命令hexo new page about，会在路径D:\Blog\source下生成about文件夹以及about文件夹下文件index.md，在index.md文件中编辑内容，编辑完成后，在D:\Blog路径，执行命令hexo clean && hexo g && hexo d上传Github

总结

突然间有点忘记该说什么了,其实总结下:
今天想说的是:

想做什么,马上就去做,而且只做这件事情.
不要想着一步做好,你要保持着永远都是一个新人的心态,新人是不可能一步就把事情做好,
想要一步登天的这种完美心态只会让你畏畏缩缩,不去做事情,
不要怕做的糟糕,管他的,只要在学习,在完善,在路上就好了.

想起来了之前学习到的一个点,
你永远只活在今天,只活在现在,不可能活在未来,
待会再做,未来再做 = 永远不会做,
所以,戒撸从此时此刻做起,锻炼从此时此刻做起,学习从此时此刻做起,
改变从此时此刻做起!
